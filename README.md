Monitoramento Veicular com OBD-II e IoT

📌 Descrição

Este projeto simula um sistema de monitoramento de veículos utilizando conceitos de OBD-II e Internet das Coisas (IoT).
Nesta versão inicial, é utilizado um sensor virtual para gerar dados de velocidade.


🎯 Objetivo
Simular dados de um veículo
Testar uma arquitetura básica de IoT
Servir como base para projetos com hardware real


🧠 Como funciona
O sistema gera valores aleatórios de velocidade e simula o envio desses dados para um sistema IoT.


▶️ Como executar
Instale o Python
Salve o código em um arquivo (ex: main.py)
Execute no terminal:
python main.py


🚀 Próximos passos
